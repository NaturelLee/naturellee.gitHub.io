# 代码块继承

.anotherDemo{ @extend .webdemo; border: solid 1px #fff; }

  1. @extend 不能继承选择器序列；
  2. 使用%,用来构建只用来继承的选择器；
$webdemo {
  margin: 10px;
}
.anotherDemo {
  @extend $webdemo;
  border: solid 1px #fff;
}

# sass 中的@media

phone tablets medium devices large devices
<768px >=768px >=992px >=1200px
@mixin col-sm($width: 50%) {
  @media (min-width: 768px) {
    width: $width;
    float: left;
  }
}
